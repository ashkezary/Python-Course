## معرفی کتابخانه‌ی numpy
برای کار با آرایه‌ها و توابع ریاضی به طور بهینه، می‌توانیم از کتابخانه‌ی `numpy` استفاده کنیم. این کتابخانه داده‌ها را در قالب `np.array` نگه داشته و مدیریت می‌کند. سازگاری با کتابخانه `pandas`، تبدیل، برش و چسباندن داده‌ها و نیز امکان کار با توزیع‌های آماری از دیگر مزایای این کتابخانه هستند.
