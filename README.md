# مبانی برنامه‌نویسی پایتون
**دانشگاه یزد**

**سال تحصیلی ۱۴۰۴-۱۴۰۳**

جزوات بالا، درون‌مایه‌ی اصلی بخش عملی درس **«مبانی برنامه‌نویسی کامپیوتر و آزمایشگاه»** بوده که در دانشگاه یزد و برای دانشجویان مهندسی مکانیک ارائه شده است. برای دیدن جزییات بخش نظری درس که آقای دکتر فرشی ارائه می‌کردند، می‌توانید [این آدرس](https://pws.yazd.ac.ir/farshi/Teaching/Mabani-Mec-4031/Mabani.html) را نگاه کنید.
