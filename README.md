# مبانی برنامه‌نویسی پایتون
**دانشگاه یزد**

**سال تحصیلی ۱۴۰۴-۱۴۰۳**

جزوات بالا، درون‌مایه‌ی اصلی بخش عملی درس **«مبانی برنامه‌نویسی کامپیوتر و آزمایشگاه»** بوده که در دانشگاه یزد و برای دانشجویان مهندسی مکانیک ارائه شده است.
